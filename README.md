## Featured Projects

### [ScoutIQ](https://github.com/Zepronex/ScoutIQ)

**Fantasy Premier League data and optimization platform**

ScoutIQ collects official football data, processes historical player records, evaluates point projections, and generates squad and transfer recommendations under Fantasy Premier League rules.

**Key features**

* REST API ingestion with validation and source tracking
* Four-stage PySpark pipeline running on Databricks
* 19 managed Delta tables built from 29,747 historical records
* PostgreSQL storage for predictions and application data
* Rule-based squad, transfer, captain, and lineup optimization
* Structured explanations with a local fallback when external services are unavailable
* 118 automated tests

`Python` `SQL` `PySpark` `Databricks` `Delta Lake` `PostgreSQL` `TypeScript`

---

### [Canvas Deadline Copilot](https://github.com/Zepronex/Canvas-Deadline-Copilot)

**A private, local assignment tracker for Canvas LMS**

A Chrome extension that collects assignments from several Canvas courses and displays them in one searchable dashboard. Credentials and cached assignment data remain on the user's device.

**Key features**

* Canvas REST API synchronization with pagination support
* Search, course filters, and deadline status filters
* Local caching and recovery when individual course requests fail
* Browser notifications and scheduled reminders
* Google Calendar and iCalendar export
* Local credential storage with limited browser permissions
* 83 automated tests

`TypeScript` `React` `Canvas REST API` `Chrome Manifest V3`
